How To Use The Simulator

I recommend to read Dr. Vogler's own description of his analog computer. To the right is an area with four button just below the Demo and Help with four buttons. They are name on/off , reset , intgr. and addr. . The on/off - button turns the analog computer on and off. The reset button resets it. The intgr. button has three different states IC , HOLD and RUN. (more on the integrator section) The addr. is of special interests as it views the current value of the different operations.

The K text input fields and Wiring

On top in the middle are the text input fields which allow you to input four floating point values. Each text input field has a name like K1, K2, K3 or K4. In order to use these values for operations like addition, multiplication etc. you have to wire them with the operators. On the left side below you see yellow and red buttons with a K and a number on them. Click on them and a greek letter and a number appears. On the right side are the "operators" like summers for addition, integrators for integration, etc. After you have clicked on one of the K buttons and a greek letter appeared you should get the same greek letter and number combination appear on one of these buttons after you clicked them. In case it was not successful , undo it by clicking and try again. Usually it works then. Connect two K buttons with a summer. Then turn on the simulator and switch to the either S1, S2 or S3 by clicking on the already mentioned addr. button. You should get the sum with a false sign. So +1 and +2 should show -3. Accept this as an oddity of analog computers.

The Integrators

Remember the intgr. button. Usually integrators have three states. They have an initial value or start value. In order to initial them the computer has to be on and you have to set the intgr. button to IC. The HOLD mode stops the integration. In RUN mode the initial value decreases by a set value.

The Oscilloscope Mode

Clicking once at the grey screen turn it into oscilloscope mode. It is now possible to get sinus curves , functions and diagramms displayed. It does not do more - probably less - than an oscilloscope.

What Is An Overload ?

Usually capacitors are used in usual analog computers. Also they operate at limited voltages and values can not exceed a certain value. The highest value in this simulator is set to -18000 and +18000.

Unimplemented Buttons or Functions

Not everything of the original Vogler Analog Computer is implemented. The square root and certain other buttons were left out or are not implemented but the majority of it should work as intended by Doctor Vogler.

known Bugs

Usually wiring and some minor other things do not quite work as expected or wanted. Please feel free to correct them.
